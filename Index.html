Inicio

Bienvenidos a nuestra página web, un proyecto desarrollado como parte de nuestro trabajo escolar sobre diseño web. En este sitio, exploraremos los conceptos fundamentales de HTML, la estructura básica que permite crear páginas en internet. A través de este ejercicio, demostramos cómo se pueden organizar los contenidos, agregar imágenes, enlaces y otros elementos esenciales para la creación de un sitio web funcional.

Este proyecto fue diseñado con código HTML simple, con el objetivo de comprender mejor su funcionamiento y aplicar los conocimientos adquiridos en clase. La estructura de la página es clara y fácil de navegar, permitiendo a los visitantes explorar diferentes secciones. Esperamos que este sitio sirva como ejemplo de lo que se puede lograr con HTML y motive a seguir aprendiendo sobre desarrollo web.
